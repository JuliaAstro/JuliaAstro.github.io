Internals

EarthOrientation.getdate — Method
getdate(data)

Determine the creation date of an IERS table by finding the last entry which is marked as "final".

EarthOrientation.update — Method
update(; force=false)

Download weekly EOP data from the IERS servers if newer files are available or no data has been downloaded previously. If the keyword force is true, then all the files will be downloaded again.
