    Implementing a server

    Create an XPA server

    To create a new XPA server, call the XPA.Server method:

    server = XPA.Server(class, name, help, send, recv)

    where class, name and help are strings while send and recv are callbacks created by the XPA.SendCallback and XPA.ReceiveCallback methods:

    send = XPA.SendCallback(sendfunc, senddata)
recv = XPA.ReceiveCallback(recvfunc, recvdata)

    where sendfunc and recvfunc are the Julia methods to call while senddata and recvdata are any data needed by the callback other than what is specified by the client request (if omitted, nothing is assumed). The callbacks have the following forms:

    function sendfunc(senddata, xpa::Server, params::String,
                  buf::Ptr{Ptr{UInt8}}, len::Ptr{Csize_t})
    ...
    return XPA.SUCCESS
end

    The callbacks must return an integer status (of type Cint): either XPA.SUCCESS or XPA.FAILURE. The methods XPA.seterror() and XPA.setmessage() can be used to specify a message accompanying the result.
